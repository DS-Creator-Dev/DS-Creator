# DS-Creator
DS Creator is a new game engine that let's you create games for the Nintendo DS.

APP
 - $ npm init -y
 - $ npm install electron
 - $ npx -p typescript tsc --init
 - $ npm i @electron/remote
 - $ npm i electron-browser-storage
 - $ npm install desmond-emu
 - $ npm start
